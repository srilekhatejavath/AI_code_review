# 🤖 AI Code Review using Gemini

## Overview
AI Code Review is a Python web application built with **Gradio** and **Google Gemini API**. It analyzes source code, identifies bugs, suggests improvements, checks coding style and security, generates improved code, and provides a rating.

## Features
- Code Review
- Bug Detection
- Performance Suggestions
- Code Style Check
- Security Analysis
- Improved Code Generation
- Code Rating (Out of 10)

## Technologies
- Python
- Gradio
- Google Gemini API

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
